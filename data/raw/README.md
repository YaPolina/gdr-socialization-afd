# Raw Data

This folder contains the ALLBUS 2023 (German General Social Survey) microdata file (`ALLBUS_2023.dta`), which is not included in the repository due to licensing restrictions.

To obtain the data, request access from:

**GESIS -- Leibniz Institute for the Social Sciences**
https://www.gesis.org

Study number: ZA8830
DOI: https://doi.org/10.4232/1.14544
